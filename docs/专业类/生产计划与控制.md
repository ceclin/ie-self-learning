# 生产计划与控制
## 课程简介
- 先修要求：基础工业工程、运筹学
- 课程难度：🌟🌟
- 推荐课时： 40课时

工业工程专业核心课程，主要讲解如何制定科学合理的生产计划，并采用严格的生产控制手段使企业有效地跟踪市场、合理地安排物料、设备、人力资源和资金等，以降低生产成本、缩短交货期和提高产品质量，提高企业运行的效率。课程内容包含：生产系统与运作战略、预测、综合生产计划、面向确定需求的库存控制、面向不确定需求的库存控制、主生产计划、JIT、运作调度等。
